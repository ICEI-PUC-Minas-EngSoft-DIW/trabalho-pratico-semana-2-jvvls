Seguidor Solar

PProjeto desenvolvido no laboratório GREEN utilizando C++ na Arduino IDE, com foco no controle automático da posição de uma placa fotovoltaica para maximizar a captação de energia solar. Além da programação do microcontrolador, participei ativamente do projeto mecânico, realizando a seleção e especificação de componentes para garantir maior eficiência e durabilidade do sistema. O resultado foi um protótipo funcional que uniu software, eletrônica e mecânica de forma integrada.

Hidroponia

Projeto pessoal criado em C++ e Arduino IDE para o desenvolvimento de um sistema autônomo de cultivo de frutas e hortaliças por meio da técnica de hidroponia. O sistema foi projetado para controlar de forma inteligente o fluxo de água e a concentração de nutrientes, otimizando o crescimento das plantas e reduzindo desperdícios. Essa experiência me permitiu explorar a aplicação prática de conceitos de automação, sensores e controle em um contexto real e sustentável.

Instagram Manager

Projeto em Python desenvolvido para atender à necessidade de um cliente que desejava gerenciar múltiplas contas do Instagram simultaneamente. Para isso, implementei rotinas utilizando web scraping e a biblioteca Selenium, permitindo automação de tarefas como login, postagem e interação com perfis. A solução trouxe ganho de eficiência e escalabilidade, reduzindo o tempo de gerenciamento manual das contas.
